# 搜种马插件

> 以前(gocq时期)给ddbot写的繁中服搜种马插件
> 由于后来转了官bot,无法验证现在插件是否可用
> 数据来源[繁中服DB](https://umatwdb.com)

指令:

- `搜种马 n因子(支持多个) 马娘名`

ps：部分因子支持别名(如春,东京,地固,逃直等),部分马娘支持外号
    示例：
    - `搜种马 10胜九力5u3地固星云`

- `支援卡列表 属性` 查看对应属性支援卡的编号
- `搜支援卡 编号 突破数(可不写)`搜索支援卡
- `uma别名 马娘名/因子名 别名`为马娘或因子添加别名,用于简化搜索
- `更新种马数据`更新数据
